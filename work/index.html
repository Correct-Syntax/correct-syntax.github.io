Design and Development Work

A selection of my work and side projects

Ozark Solar

Modern, multi-fold business card design.

Multi-fold business card design

Beau, a solar consultant for Ozark Solar, needed a multi-fold business card to give out to residents in the area that might be interested in having a solar energy system installation.

The Psalms 23 Project

Landing page website for a Psalm reading project.

Sveltekit / Tailwindcss

The Psalms 23 project aims to record high-quality readings of the Psalms in nature with the peaceful atmosphere and sounds of creation. I created a unique website for the project, and captured nature photos for use on the site.

Fellowship Finder

Fellowship finder directory web application.

Laravel / Alpine.js / Tailwindcss

EliYah Ministries needed a new directory where users could search for fellowship in their area. The project included search, listings, email notifications, a user dashboard, private messaging, and moderation system.

Velocity Lapse

Time lapse app for Android.

Flutter / Dart / Kotlin

I'm the creator, developer, and designer of Velocity Lapse: an Android app to create time lapse videos. It features a clutter-free interface, manually adjustable camera settings, two capture modes, and powerful import and export. Currently in beta early-access, it has been described by users as "the best time lapse app for Android".

OPC of Ava

New website for a life saving organization.

Wordpress / CSS

Options is a Pro-life, faith-based ministry in Ava serving the community for over 15 years. I was excited to work with them, creating a new website to make it easier for vistors to find information and recieve support at their two pregnancy center locations in Missouri.

The Options of Ava website has been featured on DesignRush's best pregnancy website designs.

Flameshot

Modern website for a popular screenshot tool.

Zola / Bootstrap

The Flameshot open-source project needed a website to reflect their easy to use screenshot tool. I worked closely with the project maintainers to tailor the website to their needs using a clean design, modern web tech, and best practices.

OpenImageIO

Logo and brand guidelines for a C++ library.

Logo / Brand guidelines

I created the logo and brand guidelines for OpenImageIO, a library for reading and writing images. The open-source library is used extensively in animation and VFX studios all over the world, and is incorporated into several commercial products.
